# 🚀 2단계 - 사다리(생성)
------------

## 기능 요구사항

* 사용자 입력 view
    * [ ] 참여할 사람들의 이름을 입력받는다.
      * [ ] 이름은 쉼표로 구분한다.
      * [ ] 이름은 최대 5글자까지 부여할 수 있다.
    * [ ] 최대 사다리 높이를 입력 받는다.

* 사다리 생성
    * [ ] 사다리 폭을 이름보다 넓게 만든다.
    * [ ] 자동으로 로또를 생성한다.
    * [ ] 라인이 겹치지 않도록 생성한다.

* 사다리 타기 로직
    * [ ] 아래 방향과 좌우 방향으로 사다리를 탄다.

* 출력 view
    * [ ] 생성한 사다리를 이름들과 함께 출력한다.

## 프로그래밍 요구사항

* 자바 8의 스트림과 람다를 적용해 프로그래밍한다.
* 규칙 6: 모든 엔티티를 작게 유지한다.
