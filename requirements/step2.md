# 🚀 2단계 - 사다리(생성)
------------

## 기능 요구사항

* 사용자 입력 view
    * [x] 참여할 사람들의 이름을 입력받는다.
        * [x] 이름은 쉼표로 구분한다.
        * [x] 이름은 최대 5글자까지 부여할 수 있다.
    * [x] 최대 사다리 높이를 입력 받는다.

* 사다리 생성
    * [x] 사다리 폭(세로선 간격)을 이름보다 넓게 만든다.
    * [x] 좌표를 랜덤으로 생성한다.
    * [x] 좌표가 겹치지 않도록 생성한다.

* 출력 view
    * [x] 생성한 사다리를 이름들과 함께 출력한다.

## 프로그래밍 요구사항

* 자바 8의 스트림과 람다를 적용해 프로그래밍한다.
* 규칙 6: 모든 엔티티를 작게 유지한다.
